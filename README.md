# ShellHN
## Requirements
Python 3.8.6 or higher.
## Getting started
1. Clone the project:  
`git clone https://github.com/yosi1314/ShellHN.git`  
2. Install packages:  
`pip install -r requirements.txt`
3. Cd to the app directory:  
`cd /app`  
4. Start the app:  
`python ShellHN.py`

## Tests
Simply run:  
`pytest`
